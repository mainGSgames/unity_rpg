    Unity implements the cross-fading effect by using either screen-space dithering or transparency.

    Enable LOD cross-fading

    To enable LOD cross-fading:

    1. In the Universal Render Pipeline (URP) Asset, enable LOD Cross Fade.

    2. Enable GPU Resident Drawer. This feature is required for cross-fading to work when using MeshThe main graphics primitive of Unity. Meshes make up a large part of your 3D worlds. Unity supports triangulated or Quadrangulated polygon meshes. Nurbs, Nurms, Subdiv surfaces must be converted to polygons. More info

    If your project uses the Built-In Render Pipeline, go to Project Settings > Quality > Level of Detail, and enable the LOD Cross Fade property.
